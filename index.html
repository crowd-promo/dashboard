Install as Desktop App
Install Crowd Promo as a standalone app on your Mac or PC. It will get its own icon, open without a browser bar, and run independently.
Manual Install Steps
Chrome / Edge: Click the install icon (⊕) in the address bar, or use the menu → "Install Crowd Promo"
Safari (Mac): File → Add to Dock
Auto-launch at startup: After installing, right-click the app icon → Options → Open at Login (Mac) or add to Startup folder (Windows)
Server Option
For full PWA features (offline support, auto-updates), host this file on any server. Free options: Netlify Drop (drag & drop) or Vercel. Once hosted, the install button will work automatically.
