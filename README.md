# Crowd Promo — Client OS

A personal client project management dashboard for Crowd Promo digital marketing agency.

## Features
- Client dashboard with time tracking, live timer, and deliverables
- Contract values, retainer tracking, and revenue vs $250k/mo goal
- Analytics page with month-over-month charts
- Resource links (Notion, Google Drive, Figma, etc.)
- Desktop notifications and email reporting
- Installable as a desktop PWA app

## Usage
Open the live site and click the install button in your browser address bar to install as a desktop app.

## Tech
Single-file HTML/CSS/JS app. No backend. Data stored in localStorage.
